# Spring Boot Exercises

This file contains interactive hands-on exercises to help you learn Spring Boot through the movieAPI project. Each exercise is designed to enhance your understanding and practical skills. 

## Exercise 1: Setting Up Spring Boot
- Create a new Spring Boot application.
- Add necessary dependencies in your `pom.xml` or `build.gradle`.
- Run the application and ensure it's working.

## Exercise 2: Building RESTful Endpoints
- Create a REST controller for managing movies.
- Implement GET, POST, PUT, and DELETE methods to interact with movie data.
- Test the endpoints using Postman or any API testing tool.

## Exercise 3: Connecting to a Database
- Set up a connection to a database (e.g., H2, MySQL).
- Implement JPA repository for movie management.
- Create a movie entity and perform CRUD operations.

## Exercise 4: Implementing Exception Handling
- Create a custom exception class.
- Implement global exception handling for your application.
- Test error messages returned by invalid requests.

## Exercise 5: Writing Unit Tests
- Write unit tests for your service and controller layers using JUnit and Mockito.
- Ensure your tests cover all edge cases and scenarios.

## Exercise 6: Deploying the Application
- Deploy your Spring Boot application to a cloud platform (e.g., Heroku, AWS).
- Ensure your application is accessible and works as expected after deployment.

## Conclusion
These exercises will guide you through the essential features of Spring Boot, helping you understand how to develop robust applications efficiently. Happy coding!
